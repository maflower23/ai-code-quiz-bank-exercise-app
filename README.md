# ai-code-quiz-bank-exercise-app
完全由AI编程，供个人练习题库使用的刷题软件。A quiz exercise/exam app for personal,completely programmed by AI.<br>
纯黑背景的夜间模式。支持解析.xlsx 与.json等格式的文件批量导入与导出题库。<br>
支持单选、多选、判断、填空（支持一题有多个填空）。题目支持插入图片。<br>
软件有三个主页面，第1个页面是用户的题库列表，可以选中题库查看练习的情况统计，浏览题库（支持题型分开浏览），题目搜索，编辑题库。<br>
第2个页面是练习/考试模式，具有单题库练习模式、单题库随机抽题考试模式（允许设置考试题目数量，若单选题多选题打乱选项）、选择多个题库并混合抽题考试模式（允许设置考题数量）、题目收藏、答题卡显示对错与跳转的功能。<br>
第3个页面是错题和收藏的题目，有练习错题记录，考试错题记录。<br>
